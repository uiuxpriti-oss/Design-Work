# Company logos

Drop the real experience logos here with these exact filenames. Each renders in
a 48×48 rounded white tile (`object-contain`), so a square or transparent PNG/SVG
works best. Until a file exists, a brand-colored monogram tile is shown instead.

| File            | Company                            | Source                          |
| --------------- | ---------------------------------- | ------------------------------- |
| `netlink.png`   | Netlink Software Group America Inc | https://netlink.com/            |
| `shanti.png`    | Shanti Infosoft LLP                | https://shantiinfosoft.com/     |
| `incscale.png`  | Incscale Technology                | https://incscale.in/            |
| `aashman.png`   | Aashman Foundation                 | https://www.aasmaanfoundation.org/ |

`.svg` also works — if you use SVGs, update the `src` paths in
`src/data/content.ts` accordingly (e.g. `/logos/netlink.svg`).
